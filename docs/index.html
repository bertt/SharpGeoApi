SharpGeoAPI

SharpGeoAPI is an implementation of the OGC API suite of standards.
The server provides the capability for organizations to deploy an RESTful OGC API endpoint using OpenAPI, GeoJSON, and HTML.
SharpGeoAPI is open source and released under an MIT license.

sample live site

Live demo: https://sharpgeoapi.herokuapp.com

Easy install with Docker

``` $ docker run -it -p 5000:80 bertt/sharpgeoapi ```
